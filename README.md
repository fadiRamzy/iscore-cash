# محرك البحث الائتماني المصري 🔍

**أكواد الاستعلام الائتماني للبنوك والمؤسسات المالية المصرية 2025**

إعداد: **فادي عادل** | كاش للتمويل متناهي الصغر | بني سويف

---

## 🚀 الميزات

- **بحث فوري** بالاسم أو الكود أو التصنيف
- **تطبيع عربي متقدم**: يتعامل مع اختلافات الهمزة، التاء المربوطة، الألف المقصورة
- **بحث ضبابي (Fuzzy Search)** باستخدام Fuse.js
- **192 مؤسسة** مقسمة على 14 تصنيف
- **تصفية** حسب النوع
- **نسخ الكود** أو **نسخ البيانات الكاملة** بنقرة
- **تصدير CSV** للنتائج
- **آخر عمليات البحث** محفوظة محلياً
- **وضع مظلم** بالكامل
- **متجاوب** مع الجوال والحاسوب
- **يعمل بدون إنترنت** بالكامل

---

## 📁 هيكل المشروع

```
credit-search/
├── index.html          ← الصفحة الرئيسية (كل شيء في ملف واحد)
├── data/
│   └── codes.js        ← قاعدة البيانات (192 مؤسسة)
└── README.md
```

---

## 🔧 تشغيل محلي

فقط افتح `index.html` مباشرة في المتصفح — لا يحتاج سيرفر.

---

## 🌐 نشر على GitHub Pages

1. أنشئ مستودع جديد على GitHub
2. ارفع جميع الملفات
3. اذهب إلى Settings → Pages
4. اختر `main` branch → `/root`
5. احفظ — سيكون الموقع جاهزاً خلال دقيقة
